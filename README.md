# Ekodrix HRMS (WorkFlow Pro)

Modern SaaS HRMS built with Next.js 14 (App Router), Supabase, Tailwind, shadcn/ui, Zustand, and Framer Motion.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Create `.env.local` (see `env.local.example`).

3. Run dev server:

```bash
npm run dev
```

## Supabase

- Run the SQL in `supabase/schema.sql` inside Supabase SQL editor.
